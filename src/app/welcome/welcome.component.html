Welcome to the EFIS editor!

This app lets you import, modify and export files in formats used by many modern aircraft EFIS avionics systems.
For full documentation, please see our GitHub Project.
Choose the type of file you'd like to work on (or you can open the navigation bar any time by clicking on the top-left "hamburger" button):

Checklists (e.g .ace, .txt, .afd, .{{ FOREFLIGHT_FILE_EXTENSION }})

Checklist files to be displayed on your EFIS screen.

Files are edited in a common interface, and then adapted for each avionics system when they're downloaded. For instance, if you download checklists for an avionics system that can only display 40 columns of text, lines will be wrapped at 40 columns.

Currently supported import/export formats:

  • Garmin G3X™/GTN™ (.ace)
  • Dynon SkyView™ - multiple width options (.txt))
  • Advanced Flight Systems (.afd)
  • Grand Rapids Technology (.txt)
  • Boeing ForeFlight (.{{ FOREFLIGHT_FILE_EXTENSION }})

Not all avionics systems support all the features of this editor.

IMPORTANT:
This app does not save your files on any server - they're only stored on your browser's local storage.
If you clear your browser's storage, or lose your computer, they'll be gone. Forever.
Be sure to download a copy of your data and keep it somewhere safe.
IMPORTANT:
Not all formats can represent entered items in the same way! Do check that the checklists are complete and appear as you expected when converting between formats.
Specifically, the ForeFlight format supports:
  • Only 3 pre-set group category types (Normal, Abnormal, Emergency)
  • Title, Challenge, Challenge/Response and Note to Challenge/Response (exactly 1 level indent) checklist item types
Other items will be silently ignored when exporting the checklist file!
